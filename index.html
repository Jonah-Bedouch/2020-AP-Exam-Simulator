AP Exam Simulator V0.1

Prepare an Exam

Welcome to the 2020 Unofficial AP Exam Simulator! This is based off of the AP Demo released by Collegeboard, and should emulate (fairly well) the examination process for any questions that you desire to test on. Upload one or two questions as a PDF and begin the exam. The timer will immediately start going. When it is complete, if there is a second question, you will be transitioned to it automatically. The submission process is identical to that of the AP Exam, with the caveat that removal of individual images from the image upload screen is not supported, unlike on the official AP Exam. It should also be noted that this page is only built to run on computers using Chrome, meaning that using this system to prep for exams where you will upload an image (through a phone) is not reccomended. The system does not retain questions or exam responses past your session (refresh), so ensure that you have a copy of everything you need on your system (IE: Don't delete your response if you need to turn it in to a teacher, and don't delete the question off of your computer until you are done responding to it).

Attach Your Response

Requirements

  • Include your AP ID and your initials at the top of the page.
  • You can attach only 1 text file per exam question.
  • Minimum file size: 1KB
  • Maximum file size: 15MB
  • Accepted file formats: .txt, .doc, .docx, .pdf, .odt
  • NOTE: This practice system does not enforce file size.

Requirements

  • If your response is longer than 1 page, you can attach up to 5 photos
  • Take 1 photo for each page, write only on 1 side, and make sure the entire page is in the shot
  • Write your AP ID, your initials, and the page number at the top of each page.
  • You can only attach 1 photo at a time. Do not hit "Submit" until you've attacked all your photos.
  • Attach your photos in order (page 1, then page 2, etc.).
  • Accepted file formats: .png, .jpg, .jpeg
  • Minimum file size: 10KB
  • Maximum file size: 25MB
  • NOTE: This practice system does not enforce file size.

Paste Your Response

Draft your response in a different application, then paste it here. You cannot type directly into the field.

  • Include your AP ID and your initials at the top of the page.
  • If you make a mistake, hit "Clear Field", and try again.
  • Check the formatting and content in your pasted response. If it's wrong or incomplete, hit "Clear Field", go back to your source doc, correct the issue, and copy and paste again.
  • When you're done, hit "Submit"
